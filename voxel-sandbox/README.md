# Voxel Sandbox

A small Minecraft-like sandbox made with TypeScript, Bun, Vite, and Three.js.

## Run

```bash
bun install
bun run dev
```

Terminal mode:

```bash
bun run start -- -i
```

## Controls

Browser:

- Click **Enter world** to lock the pointer.
- Move with `W`, `A`, `S`, `D`.
- Jump with `Space`; sprint with `Shift`.
- Look with the mouse.
- Left click removes the targeted block.
- Right click places the selected block.
- Select blocks with number keys `1` through `7` or the mouse wheel.
- Press `R` to regenerate the world.

Terminal:

- Move with `W`, `A`, `S`, `D`.
- Turn with the left and right arrow keys.
- Look up/down with the up and down arrow keys.
- Press `B` to break the targeted block.
- Press `P` to place the selected block.
- Select blocks with number keys `1` through `7`.
- Press `R` to regenerate the world.
- Press `Q` or `Ctrl+C` to quit.
